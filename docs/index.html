Setup in 30 seconds

# 1. Sign up (one time)
curl -X POST https://gabe.usegabe.workers.dev/api/signup \
  -H "Content-Type: application/json" \
  -d '{"email":"you@example.com"}'

# 2. Create a monitor (one time)
curl -X POST https://gabe.usegabe.workers.dev/api/monitors \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{"name":"Nightly backup","interval_minutes":60}'

# 3. Add to your crontab (done forever)
0 * * * * /usr/bin/backup.sh && curl -fsS gabe.usegabe.workers.dev/ping/YOUR_ID

How it works

1

Create a monitor

Tell Gabe what job to watch and how often it should run. Get a unique ping URL.

2

Add the ping

Append curl gabe.usegabe.workers.dev/ping/ID to the end of your cron job.

3

Get alerts

If your job doesn't ping on time, Gabe emails you immediately. No noise, just the alert.
