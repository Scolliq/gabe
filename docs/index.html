API Reference

Base URL: https://gabe.usegabe.workers.dev

POST /api/signup

Create an account and get your API key.

Body: {"email": "you@example.com"}
POST /api/monitors

Create a new monitor. Returns a ping URL.

Body: {"name": "My job", "interval_minutes": 60}
GET /ping/:id

Send a heartbeat. Add this to the end of your cron job.

GET /api/monitors

List all your monitors and their current status.

DELETE /api/monitors/:id

Delete a monitor.
